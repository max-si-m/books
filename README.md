# Deploy to Heroku
You need to sign up in heroku.com

Login in CLI

```
$ heroku login
```

Create new app:

```
$ heroku create
```

Push to heroku(via git):

```
$ git push heroku master
```
Run migrations:

```
$ heroku run rails db:migrate
```
Install seeds:
```
$ heroku run rake db:seed
```

Well done.
